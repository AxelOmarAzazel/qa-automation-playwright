# QA Automation Playwright

Proyecto de aprendizaje de automatización de pruebas utilizando Playwright y TypeScript.

## Tecnologías

* Playwright
* TypeScript
* Git
* GitHub

## Casos de prueba implementados

- Validación de título
- Búsqueda en documentación
- Navegación a Locators
- Implementación de Page Object Model
- Uso de Hooks

## Objetivo

Desarrollar habilidades de QA Automation y construir un portafolio profesional de pruebas automatizadas.
